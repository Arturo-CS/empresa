# Laboratorio - Validación de Formulario

## Description

Validación de formulario para registrar correctamente una persona usando el ORM Prisma.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
